# Мобильная версия приложения нейроинтерфейса EasyEEG BCI
Для смартфонов и планшетов на OC Android

## Версии:
**v.0 - EasyEEB-BCI-9.apk**

Возможности:
- Получение ЭЭГ с устройств EasyEEG BCI
- Фильтрация сигнала ЭЭГ
- Визуализация ЭЭГ
- Спектральный анализ ЭЭГ в реальном времени по основным частотным диапазонам
